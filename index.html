Programs Written

Repositories hosted on Github.

  • phylip2eigenstrat: Python script to generate .snp and .ind Eigenstrat files from a PHYLIP infile
  • extractVariantSites4raxml: Python script for counting and removing invariant sites from a PHYLIP file for use with the RAxML ascertainment bias correction
  • makePopMap: Python program to add population identifiers to STRUCTURE, PLINK, and PHYLIP files. Also can just output a tab-separated file with sample names and population IDs. Population IDs are identified from sample names.
  • coordinate_converter: Python program to convert GPS coordinates from one format to another (currently supports degrees/minutes/seconds, degrees decimal minutes, and decimal degrees). Can take mixed coordinate formats in a file.
  • ddrad_scripts: Various scripts for dealing with ddRAD data, mostly dealing with output from pyRAD.
  • bash_scripts: Various bash scripts for working on the AHPCC computer cluster.
  • R_scripts: Currently, there is an R script to parse Patterson's D-test output generated using a colleague's C++ program (Comp-D by smussmann82). The other two R scripts are for parsing Microsoft Excel files in a format specific to the lab I am in (Douglas Lab).
  • file_converters: File conversion scripts for converting various DNA sequence file formats to other formats.
  • parallelMrBayes: Perl program to run MrBayes on multiple loci in parallel. Also automatically writes a MrBayes block into each NEXUS file.
